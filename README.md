### `remotecache.vdf` generating guide: 
#### (in case game goes back to main menu or they don't show up)

### Step 1: Close Steam
* **Shut down Steam completely.**
* Check your taskbar tray to ensure it is fully closed.

### Step 2: Place Your Resigned Saves
1. Run your 007 Save Tool to resign your saves.
2. Put the new save folders (containers) into this exact folder:
   `\Steam\userdata\<Your_Account_ID>\3768760\remote\`

### Step 3: Generate the VDF File
1. Download and unzip the **[RemoteCacheVdfGenerator](https://github.com/mi5hmash/RemoteCacheVdfGenerator/releases/download/v1.0.0/win-x64_v1.0.0.zip)**. (*it's from mi5hmash so kudos to them*)
2. Copy `remote-cache-vdf-generator.exe`.
3. Paste it directly into the **3768760** folder (right next to the `remote` folder).
4. Double-click the `.exe` file to run it. 

*Note: A new `remotecache.vdf` file will instantly appear in the folder.*

### Step 4: Play
1. Open Steam.
2. Launch 007. Your saves will now load properly.
